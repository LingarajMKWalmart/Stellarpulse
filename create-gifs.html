📹 How to Create GIFs for Contest Submission

  1. Use a screen recording tool like Loom, OBS, or macOS QuickTime Player
  2. Record each scene for 5-8 seconds to capture the full animation cycle
  3. Convert to GIF using online tools like ezgif.com or CloudConvert
  4. Optimize file size - Keep under 3MB for contest submissions
  5. Recommended dimensions: 800x600px for best quality/size balance
